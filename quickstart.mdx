# Quick Start

## Install

```bash
npm install @owizdom/config-validator
# or
pnpm add @owizdom/config-validator
```

## Validate your OApp

Pass one or more deployments with `-d chain:address`:

```bash
npx lz-config-validator validate \
  -d ethereum:0x1234567890abcdef... \
  -d arbitrum:0xabcdef1234567890... \
  -d optimism:0x9876543210fedcba...
```

Example output:

```
Validating OApp across 3 chains (6 pathways)

  ✓ [ethereum -> arbitrum] peers: Peers set for ethereum <-> arbitrum
  ✓ [ethereum -> arbitrum] dvns: DVN sets match for ethereum <-> arbitrum
  ✓ [ethereum -> arbitrum] executor: Executor set on arbitrum
  ✓ [ethereum -> arbitrum] confirmations: Confirmations OK (outbound 15 >= inbound 15)
  ...

Summary: 6/6 pathways valid, 0 errors, 0 warnings
```

If there are errors, use fix suggestions:

```bash
npx lz-config-validator validate \
  -d ethereum:0x... -d arbitrum:0x... \
  --fix-mode=report
```

## Use the SDK

```ts
import { validateOApp, printReport, getFixes } from '@owizdom/config-validator';

const report = await validateOApp({
  deployments: [
    { chain: 'ethereum', address: '0x1234...' },
    { chain: 'arbitrum', address: '0x5678...' },
  ],
});

printReport(report);

if (report.errorCount > 0) {
  const fixes = getFixes(report);
  for (const f of fixes) console.log(f.description);
}
```
